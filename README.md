# Title Page Creator for BBSBEC Practical Files

Create professional title pages for your practical file submissions at BBSBEC.

## Getting Started (Windows)

1. **Download this project** - Extract the ZIP file to a folder
2. **Run INSTALL.BAT** - Double-click to install everything needed
3. **Run START.BAT** - Double-click to launch the application
4. **Open your browser** - Go to http://localhost:3000

That's it! Fill in your details and generate your title page.

## Getting Started (Mac/Linux)

1. Install Bun: `curl -fsSL https://bun.sh/install | bash`
2. Install dependencies: `bun install`
3. Start the app: `bun run dev`
4. Open http://localhost:3000

## For Programmers

Built with Next.js + React + Bun. Key files:
- `app/` - Next.js pages and routes
- `components/` - React components
- `lib/` - Utilities and helpers
- `data/` - Static data files

Run `bun run dev` for development server. Modify components to customize the title page layout and styling.
